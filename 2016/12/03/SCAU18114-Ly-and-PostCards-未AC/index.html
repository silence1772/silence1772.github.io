SCAU18114 Ly and PostCards(未AC)

ACM

18114 Ly and PostCards

该题有题解

时间限制:1000MS 内存限制:65535K
提交次数:0 通过次数:0

题型: 编程题 语言: 不限定

Description

Long time ago, Ly was trafficked to Morocoo for Watching Finals by two big big bears.
As usual, Ly decided to send postcards to his friends after contest. However, postcards in Morocoo
are too expensive so only some lucky ones will be chosen to get it.
But as we all know, relationships in our world are complicated. That a girl called Siwa get the pos
tcard, may make another girl called Emma unhappy if she doesn’t get it.
To avoid trouble, Ly choose the lucky ones according to these steps:

1. Numbered his friends from 1 to N
2. Ramdomly choose K different ones from them into a list
3. If someone A in the chosen list will cause someone B outside the list unhappy, add B to the list
4. Repeat step 3 until nobody will be unhappy

Now, Ly do not know how many postcards he needs to buy on average(the expectation). But Ly hates floating point number very very much,
you should compute the answer * C(n, k) . C(n, k) means the number of ways to choose k elements from n different elements.

输入格式

The input file begins with an integer T (T <= 10) in one row indicating the number of test case. Then T test cases follows.
Each test case begins with three integers n, m and k (1 <= n <= 50, m <= 10000, 0 <= k <= n).
Then m lines follow.
Each line contains two integer u, v (1 <= u,v <= n), meaning that if people with number u gets a postcard, and people with number v does not get, v will be unhappy.

输出格式

Print the answer mentioned above in one line for each test case.

输入样例

3
2 1 1
1 2
2 2 1
1 2
2 1
3 3 2
1 2
2 1
1 3

输出样例

3
4
9

作者 201330330101

题意

题目大意：要发明信片给朋友，随机选择k个人进入要发送的名单，同时给出了m条关系u，v。如果u得到了明信片，那么v就会不开心，就不得不也发给v，如此直到在名单外的人不会不开心为止。问所有的情况要在名单中的总数。
例如：
3 1 2
1 2
则随机选择k个人有：
1 2
1 3
2 3
其中第二个中的1进入名单会使2不开心，因此2也必须进入名单，则情况变成：
1 2
1 3 2
2 3
总共有7个，答案为7.
