18091 Jason请大家吃糖

该题有题解

时间限制:1000MS 内存限制:65535K
提交次数:62 通过次数:18 收入:44

题型: 编程题 语言: 不限定

Description

众所周知，糖果有很多的种类，现在有个叫ion的家伙，很喜欢吃糖，于是Jason就在旁边开了间超市卖糖果，
有一天他开开心心跑到超市去买糖。看到超市里有各种各样的糖果他都方了。一方之下他怒买了n种糖果，每种都只买了一颗。
即每颗糖都是长得不一样的。回到家里他看着糖果就在想，假设把这n种糖果分堆，有多少种方案呢？不过他看着一堆糖果已经方了，
现在他需要大家的帮助，假设每堆至少1颗糖，至多3颗糖，分的堆数不限，问有多少种不同的分堆方案。

输入格式

题目有多组测试数据，用while (scanf(“%d”, &n) != EOF){} 结束读入。
每行输入一个n（0 <= n <= 20）,表示有n颗不同的糖果。

输出格式

输出可能的方案数，每组输出占一行。

输入样例

1
2

输出样例

1
2

提示

注意：分好了的堆与堆之间是无序的

来源 Jason

作者 20143033023

抄袭过程

这位出题人不行啊，直接改题，让我直接抄答案了
原题：
将n个人分成若干队，每队人数可以是1、2或者3人。

已知n，求组队方案数。
Input
输入包含多组测试数据(约1000组)，每组数据占一行，包含一个数字n(0<=n<=20)，表示ACM集训队的队员人数；n为0，表示输入结束。
Output
请输出n个队员所有可能的组队方案数，每组输出占一行。
Sample Input
1
2
3
4
5
0
Sample Output
1
2
5
14
46
记dp[i]为i个人分组的方案数。显然dp[1]=1,dp[2]=2,dp[3]=5。i>=4时，考虑最后一个人，他可以一人一队，对方案数的贡献是dp[i-1]；也可以和另外i-1人中的一人组队，贡献是C(i-1,1)xdp[i-2]；还可以和另外i-1人中的两人组队，贡献是C(i-1,2)xdp[i-3]，所以dp[i]=dp[i-1]+C(i-1,1)xdp[i-2]+C(i-1,2)xdp[i-3]。注意数据量会爆int，要用long long。
